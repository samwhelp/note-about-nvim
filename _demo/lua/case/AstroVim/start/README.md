

# AstroVim


## Link

* [AstroVim](https://github.com/kabinspace/AstroVim)


## Install

run (Script: [install.sh](install.sh))

``` sh
git clone https://github.com/kabinspace/AstroVim.git ~/.config/nvim
```


## Uninstall

run (Script: [remove.sh](remove.sh))

``` sh
rm ~/.config/nvim -rf
rm ~/.cache/nvim -rf
rm ~/.local/share/nvim/ -rf
```
