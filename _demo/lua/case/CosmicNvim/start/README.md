

# CosmicNvim


## Link

* [CosmicNvim](https://github.com/CosmicNvim/CosmicNvim)


## Install

run (Script: [install.sh](install.sh))

``` sh
git clone https://github.com/CosmicNvim/CosmicNvim.git ~/.config/nvim
```


## Uninstall

run (Script: [remove.sh](remove.sh))

``` sh
rm ~/.config/nvim -rf
rm ~/.cache/nvim -rf
rm ~/.local/share/nvim/ -rf
```
