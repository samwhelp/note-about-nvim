

# lvim-tech


## Link

* [lvim](https://github.com/lvim-tech/lvim)


## Install

run (Script: [install.sh](install.sh))

``` sh
git clone https://github.com/lvim-tech/lvim.git ~/.config/nvim
```


## Uninstall

run (Script: [remove.sh](remove.sh))

``` sh
rm ~/.config/nvim -rf
rm ~/.cache/nvim -rf
rm ~/.local/share/nvim/ -rf
```
