

# NvChad


## Link

* [NvChad](https://github.com/NvChad/NvChad)


## Install

run (Script: [install.sh](install.sh))

``` sh
git clone https://github.com/NvChad/NvChad.git ~/.config/nvim --depth 1
```


## Uninstall

run (Script: [remove.sh](remove.sh))

``` sh
rm ~/.config/nvim -rf
rm ~/.cache/nvim -rf
rm ~/.local/share/nvim/ -rf
```
